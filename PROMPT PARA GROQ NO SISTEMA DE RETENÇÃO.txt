PASSO 1 — Vercel Function
Criar arquivo: /api/improve-text.ts
================================================================

import type { VercelRequest, VercelResponse } from '@vercel/node';

export default async function handler(req: VercelRequest, res: VercelResponse) {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { text } = req.body;

  if (!text || text.trim().length < 10) {
    return res.status(400).json({ error: 'Texto muito curto para melhorar' });
  }

  if (!process.env.GROQ_API_KEY) {
    return res.status(500).json({ error: 'Chave da API não configurada' });
  }

  try {
    const response = await fetch('https://api.groq.com/openai/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.GROQ_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        model: 'llama-3.3-70b-versatile',
        max_tokens: 1024,
        messages: [
          {
            role: 'system',
            content: `Você é um assistente de escrita especializado em registros escolares
institucionais brasileiros. Você é um funcionário do Colégio Objetivo. 
Sua função é melhorar textos escritos por atendentes e coordenadores de escola,
tornando-os mais claros, formais e bem estruturados.
Regras obrigatórias:
- Mantenha TODAS as informações originais sem inventar nada
- Use linguagem formal e profissional em português brasileiro
- Corrija erros gramaticais e de pontuação
- Melhore a estrutura e clareza do texto
- Mantenha o mesmo sentido e tom institucional
- Responda APENAS com o texto melhorado, sem explicações, sem comentários,
  sem aspas`
          },
          {
            role: 'user',
            content: `Melhore este texto de registro escolar:\n\n${text}`
          }
        ]
      })
    });

    if (!response.ok) {
      throw new Error(`Groq API error: ${response.status}`);
    }

    const data = await response.json();
    const improvedText = data.choices[0]?.message?.content?.trim();

    if (!improvedText) {
      throw new Error('Resposta vazia da IA');
    }

    return res.status(200).json({ improvedText });

  } catch (error) {
    console.error('Erro na API do Groq:', error);
    return res.status(500).json({ error: 'Não foi possível melhorar o texto agora' });
  }
}

PASSO 2 — Componente Reutilizável
Criar arquivo: /src/components/AITextEnhancer.tsx
================================================================

Crie um componente que recebe as props `value`, `onChange` e `placeholder`
e renderiza um textarea completo com o botão de IA integrado.
O botão deve ficar posicionado no canto inferior esquerdo do textarea,
dentro da borda do campo.

O componente deve:

- Renderizar o textarea normalmente com todas as props passadas

- Exibir o botão "✨ Melhorar com IA" no canto inferior esquerdo com:
  * Tamanho de fonte pequeno (text-xs)
  * Cor roxa/violeta suave
  * Sem fundo, apenas ícone e texto
  * O botão só aparece quando o textarea tiver mais de 20 caracteres digitados

- Ao clicar no botão:
  * Desabilitar o textarea
  * Trocar o botão por spinner animado com texto "Analisando..." na cor roxa

- Chamar POST /api/improve-text com o texto atual

- Se sucesso:
  * Substituir o texto pelo texto melhorado
  * Exibir abaixo do textarea um card cinza claro colapsável com o texto
    original e botão "↩ Restaurar texto original"

- Se erro:
  * Exibir toast de erro "Não foi possível melhorar o texto. Tente novamente."
  * Não apagar o texto original

- O card com texto original some automaticamente se o usuário restaurar
  ou se editar o texto manualmente após a melhoria

INSTRUÇÕES PÓS-IMPLEMENTAÇÃO
================================================================

Após o Antigravity implementar, siga estes passos:

1. Pegue sua chave do Groq em:
   https://console.groq.com → API Keys → Create API Key

2. Adicione no .env.local localmente:
   GROQ_API_KEY=sua_chave_aqui

3. Adicione na Vercel em:
   Settings → Environment Variables → GROQ_API_KEY
   Marcar os ambientes: Production + Preview + Development

4. Faça Redeploy na Vercel:
   Deployments → três pontinhos → Redeploy

5. Teste nos 3 campos do sistema

OBSERVAÇÕES TÉCNICAS
================================================================

- O modelo usado é: llama-3.3-70b-versatile (Groq)
